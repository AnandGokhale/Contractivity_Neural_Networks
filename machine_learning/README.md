# Contracting Implicit Neural networks

A PyTorch implementation of implicit-layer neural networks with guaranteed contractivity, applied to image classification on MNIST and CIFAR-10.


## Overview

This code is a numerical implementation for the machine learning experiments in our CDC 2026 Submission. 



The architecture consists of:
1. A CNN feature extractor (trunk) that processes the input image
2. A parameter generator that produces the matrices defining the implicit layer
3. An implicit solver that finds the fixed point via iteration
4. A linear output head

## Architecture

### Implicit Layer

The core of the model solves for a hidden state `w` satisfying:

```
w = activation(W(x) @ w + b(x))
```

where `W(x)` and `b(x)` are input-dependent matrices/vectors generated by the network. Contractivity of the operator is enforced structurally via the parameterization of `W`, described in the paper.

### CNN Trunk

A lightweight depthwise-separable CNN processes 32×32 input images:

```
Input (C, 32, 32)
  → Conv + BN + ReLU + MaxPool    [C → 32, 16×16]
  → DepthwiseSep + BN + ReLU + MaxPool  [32 → 64, 8×8]
  → DepthwiseSep + BN + ReLU + MaxPool  [64 → 64, 4×4]
  → Flatten → Linear(1024, hidden_dim)
```


## Training Details

**CIFAR-10:**
- Optimizer: SGD with momentum 0.95, weight decay 5e-4
- LR schedule: OneCycleLR (peak 1e-2, linear annealing)
- Gradient clipping: max norm 1.0
- Batch size: 128

**MNIST:**
- Optimizer: Adam, lr=1e-3
- LR schedule: StepLR (step 10, gamma 0.5)
- Batch size: 128
